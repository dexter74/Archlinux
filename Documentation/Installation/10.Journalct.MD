**Afficher la log**
```
clear ; journalctl | grep -i "fail\|err\|warn" | tail -n 50 ;
clear ; journalctl -f -e ;
```

**Limiter la log à 24 h**
```
journalctl --vacuum-time=24h ;
```

```

# Avahi.service
systemctl start avahi-daemon.service ;
```
