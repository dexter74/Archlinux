### Démarrage du le LiveCD :


**Passer le Clavier en Azerty**
```bash
loadkeys fr-pc ;
```

**Se connecter au point WI-FI**
```bash
iwctl --passphrase <Clé WPA> station <Interface> connect <SSID> ;
```
